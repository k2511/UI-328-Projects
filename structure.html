HTML Notes

1. What is HTML?

HTML stands for HyperText Markup Language. It is used to create web pages and web applications.

2. Basic Structure of HTML


<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

3. Common HTML Tags

  • <h1> to <h6> — Headings
  • <p> — Paragraph
  • <a> — Anchor (Link)
  • <img> — Image
  • <ul>, <ol>, <li> — Lists
  • <div> — Division (block)
  • <span> — Inline element
  • <br> — Line break
  • <hr> — Horizontal rule

4. Forms in HTML

Used to collect user input.


<form action="/submit" method="post">
  <input type="text" name="name" />
  <input type="submit" value="Submit" />
</form>

5. HTML Attributes

Attributes provide additional information about elements.

  • href — For links
  • src — For images
  • alt — Alternate text
  • id, class — For styling and scripting
  • style — Inline CSS

6. Semantic Elements

Elements that clearly describe their meaning:

  • <header>, <footer>
  • <article>, <section>
  • <nav>, <aside>
  • <main>, <figure>

7. HTML Media Tags

  • <img> — Images
  • <video> — Video files
  • <audio> — Audio files
  • <iframe> — Embed external pages

8. HTML Tables


<table>
  <tr><th>Name</th><th>Age</th></tr>
  <tr><td>John</td><td>25</td></tr>
</table>

9. HTML Comments

Used to add notes within code:

<!-- This is a comment -->

10. HTML Best Practices

  • Use semantic tags
  • Always close tags properly
  • Keep structure clean and readable
  • Use external CSS/JS for styling and scripting
  • Validate HTML using W3C Validator
